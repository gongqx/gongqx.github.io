# 查理说

  1. 你的最佳策略就是让对手按照规定透露出他们的叫牌策略，然后再选用针对策略来应对。
  2. 倘若自己都不为自己活出自己的人生，那还有谁会为自己而活呢
  3. 每个职业最终都是蒙骗外行人的勾当。

# 自言自语

  1. 我不能总是全仓某一只股票，我应该学会仓位管理，二分之一，四分之一也应该用到。
  2. 应该学会逐步验证，逐步买入。预期会涨而买入的股票，如果期望落空，那就应该卖出
  3. 选那些热门的，势头足的。再根据大盘趋势操作。
  4. 该努力时高瞻远瞩 该抉择时鼠目寸光 基本废
  5. 最好的方式就是不要让人感到厌恶自己，以自己最大的能力真心实意的去帮助别人。做一个甘心付出，不求名利的人。这样是对自己真的有帮助。
  6. 一个很充实的人，有目标，有方向，知道自己想干什么。人越缺什么，越想找回什么，没有钱的时候得不到尊重，希望赚钱被别人高看一眼。这是不对的。想想钱真正能帮你做什么？
  7. 你的信仰呢，责任呢？你应该扛起什么样的责任，挑起什么样的担子，你的志向是什么？
  8. 你知道自己想要什么吗？不同的阶段有正确的策略吗？能够克服欲望，能够自律吗？
  9. 看到引人深思的文章，多些自我反省。‘
  10. 一座座坟莹都提示着生命的每一分钟都弥足珍贵。
  11. 人后的苦尚且还能克服，人前的尊严却无比脆弱。
  12. 看看那些身处命运的漩涡的人，耗尽心力去争取那些可能本就是稀松平常的东西，自己的灵魂会感到不安吗？
