# 释怀（草稿）

我走了很多的弯路，也使出了不少的力气，也埋怨了自己很长时间。可是最后发现，我丢失掉了自己的快乐，为自己的四周筑起高墙。内心一直压抑着欲望，得过且过成了生活的主旋律。最后得到的是日渐稀少的自信和与日俱增的痛苦。

我出生在一个并不贫穷的家庭里，父母都是读过几年书的人。爷爷当过兵，在乡镇政府上班，也是挺有名气的一个人。奶奶在我上小学的时候刚从教师岗位上退休。在村子里比起来家庭情况算是挺不错。

  • 追溯自己脑袋里最早的回忆，是一个夏天，我和妈妈在院子里吃晚饭，妈妈在厨房里盛饭，那个时候爸爸不在，他应该是在鄄城上班。我估摸着也就三四岁的样子吧。这个记忆是证明自己开始懂得生活这个东西了吗？小时候妈妈总把好的给我吃，自己舍不得吃。现在也是。我也是没有回报过的。

  • 我还记的爸爸和妈妈吵架那一次，我可能也很小，那时妈妈在哭，爸爸的脾气大的不得了，手指向天，像是很决绝的样子。具体说的是啥，我不记得了。我爸爸以前是个不努力的人，在酒厂上班，然后又花钱考大货车的驾照，也没去做大货车司机。妈妈每次去姥姥家回家的手，姥姥都会给我妈妈钱，或者带些东西回去。这个记忆是真实的。这个是痛苦的回忆，可能对爸爸的气愤这里就是根源了。

  • 和我哥哥姐姐以及邻居的小伙伴的记忆是，我在别人家拉粑粑，然后小伙伴的奶奶也准备方便，我就拉完出去了。不知道为什么这个记忆那么深刻，可能是叫尴尬的这种情绪活动第一次在我脑子里产生吧。

  • 还有就是照片了，大爷给我照照片，以前可喜欢照照片了，大爷经常拍。几乎所有童年老照片都是大爷拍的。经济条件好，还经常给我鸡腿吃。真的感谢大爷和大娘，现在离得远了，以前在老家就挨着。等我有钱了再回报你们吧。

  • 爷爷奶奶那里记的经常有好吃的就去叫我吃，给我打电话。或着跑过来叫我去吃。

  • 姥姥姥爷更不用说了，是很喜欢我这个外甥的，每次去姥姥家，都会得到些什么往家拿，或者吃好吃的。这种回忆是永远存在的，永远永远。姥姥让我知道父母对子女有太多的爱也是我们家花他们的钱最多，去年姥姥去世了，姥姥的爱是无私伟大的，我欠下无比巨大的情分的，几乎没有还过。想想就要落泪了，可是从姥姥去世，我从来没有梦到过她。她被我想到的时候也很少了。斯人已逝，怀念姥姥。尽力照顾好姥爷吧。缺少感恩，因为几乎没有回报过。

  • 1993.12.27生日，这样算下来是7岁开始上学，学前班，2000年上一年级，2002年二年级的时候复读了一年。所以之后显得我的年纪在班里是比较大的，一是因为距离1994年还有5天才出生，所以就按照94年出生的小孩来上学的，再有就是二年级复读了。这样看起来要大了两岁。（真悲剧）

三个村子组成的希望小学里，每个年级大概三十几个学生。，我的学习成绩算是不错。因为小的时候可以读到哥哥姐姐的书，因为奶奶曾是学校教师的缘故。老师都比较照顾我。我不记得从爸妈或者爷爷奶奶有教给我书本上的知识。当时应该是在虚荣心的驱使下，自己才想学习的吧。但是我在二年级的时候，留级了，我想不起来是因为什么原因了。但是从正常逻辑来说，那就是学的不好，跟不上班，所以才被要求留级的。到了第二个二年级的时候，我的名次就比较靠前了。从那个时候开始我就算是个优秀的学生了。我不是个调皮捣蛋的孩子。性格比较好，喜欢交朋友，和朋友们玩的都不错。

  1. 对于学校的回忆，我记得有个人问我爸爸有几个兄弟，我说是有两个，大爷和四叔，二大爷早逝了。零散的听说是先天性心脏病。
  2. 学前班老师，因为冤枉我骂人这件事，最后我在耳朵被扯掉之前屈服了。我记忆犹新啊，这辈子也不会忘记了。给我留下的是什么影响？屈服，冤枉，正义？可能也建立了自己的人格，不能做坏事，现在我准备积极的看待了，这给我的教育是积极的。虽然她错怪了我。
  3. 我时常想起学前班的时候，小伙伴们打算给自己取一个动物的名字，阿龙，阿虎，我好像是被分配成了大象，我记得我想叫阿鹤，可是合成就叫这个名字了，我就比较尴尬了，他叫阿鹤是理所应当的。我就叫阿象吧。还有煤球炉子的取暖，和扫雪活动。值日时总把家里的扫把弄坏。
  4. 我和刘步龙是挺好的朋友，因为学习都很好的原因吧，小学竞赛也都是我们两个。我经常去他家学习。以前是真的好朋友，可是到了五六年级，因为奇怪而又无法解释的事情让我们的友谊消失了。我们俩被指派为小组长，经常用书本打孙良广的脑袋，他学习不好，总是说话，我们经常教育他，记得这是我最早开始欺负别人的行为。
  5. 还有我和孙选夏背古诗的记忆。 未完待续
