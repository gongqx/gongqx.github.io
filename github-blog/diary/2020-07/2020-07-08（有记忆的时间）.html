# 2020-07-08

同样，今天一天也过得很快，但是没有收获。

我已经立志了，剩下的就是行动起来了。

我买的英文版《图灵传》也已经快递到了，我拿到手里，可惜没有几个字能看懂。

我迷上了图灵，这是我内心最深处的呐喊，我能感受到，因为别的东西对我来说，也就是浮云了了。但是我年纪已经大了。时间还在飞快的流逝。我知道这是个非常长的道路，非常长，但我已经决定要走这条路了，能走到什么样的程度，随他便了， 我只管走路吧。

2000年图灵奖得主姚期智说，AI领域或将诞生下一个牛顿或爱因斯坦。我认为就是我宫庆鑫了。

你应该知道鸭子定律，水面的上鸭子看上去悠闲自在，那是因为水下的双蹼在划水。因为你必须十分努力，才能看起来好不费力。

我必须整合自己所知道的知识，去为计算机科学学习提供帮助。

用上一切所有资源。 去完成你的目的。

我应该有科学的计划和道路，去完成实际的工作，然后才能得到想要的成果，不可以没有计划，将所有东西胡乱的堆砌。聪明人都有自己的学习方法，你也应该逐渐去摸索适合自己的那种方式。
