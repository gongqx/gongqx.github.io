# 2020-07-06

初步五年（2020-2025）计划最好的路线：

1、做一个平凡的人。（婚姻关系，亲情照顾，父母养老，孩子成才，满足虚荣）

2、拥有一份完美的工作。（7月17日之前找到一个工作，7000+）

3、做自己领域的技术大牛。（今年就安心的学习web技术，不在管自学考试的事情了，先稳住自己）

4、牺牲娱乐，弥补差距。（参加明年的自学考试，争取2年半的时间拿到毕业证）

5、追逐地平线升起的太阳。5年后（2025），你可能已经和别人站在同一起跑线上了。剩下的就是超越别人。机器学习（网络安全），这个当做是爱好，但是要排在前四个之后，不可以喧宾夺主。（这个真的是很难看到效果，因为没有人会去聘请一个专科生和普通本科生去做机器学习方面的研究，只能是在应用中的简单应用，依照别人的模型做些简单的工作而已。但是AI这支火箭从现在开始直冲云霄了，也许你可以帮助你的孩子做些有意义的事情，编舟吧，开始大渡海）

真的，其余都是美丽的泡沫。

如果考试我要学习，Java，C++,JavaScript,Python

真的是任重而道远，这足够你一直做下去的了，毕竟你不是科班出身，想要有所作为真的是非常非常的难。时间也不会等你的，你的年纪在某一时刻就会变成负担，除非你可以有真突出的地方。上面的五件事已经是非常非常的难做了。不该有别的想法了。例如：游戏、区块链、股票、一些乱七八糟的事情，这些都会让你更难，花更长时间接近目标。必要再流连忘返。做一一只昂贵的警犬。不为乱象所动。

任何时候别忘了这是你选择的路，毕竟也许有更容易走的，做一个制图员，平面设计师。在自己的能力范围内做出超越自己的事情，那就是英雄了。

我又想都了，网络安全，这件事也是很重要的，因为强国之间都会有信息战。但是现在也没有必要想清楚，因为第一二三四还没有做好呢，这是太遥远的事情。以后在想吧。如果前四个做的好，这两个都有可能做，如果做不好，哈哈，抱歉，你的生命只有一次旅程。先做个人，再做个英雄，神仙的称号不是自己封的。

初步计划，想要做机器学习，网络安全方面的研究，起码还要5年时间，那就32岁了，

5年时间，学习英语（5年间应该有所收获的），多敲代码，修身养性。

你还有很多年可以赶超一个人，宁静致远。
