# 2020-06-22

今天还是不学习，有一天你会开始学习的吧？！

今天想到了一件事要做，就是现在开始弹吉他，既然没有事情做，会特别心烦，所以就去弹吉他吧。毕竟也是很好的事情，每天固定弹上一个小时或者更长的时间，就像你现在这样，每天写一写日记。记录一下你天的工作，只要坚持下去，会有很大的收获。如果从现在开始就坚持不断的弹吉他，那以后的技术肯定会很好的，到那时候，说不定就可以找个音乐方面的工作了呢。眼下的事情还是要按照现在的去做哦

妈妈昨天晚上到了2点半还没有睡着，现在她的失眠症又犯了，是因为压力的原因。也就是我的原因。

我应该清楚以后要怎么做了，不能再只在乎自己的感受了。

在哔哩哔哩看到了一个up主（建国是一只柴柴），她说自己是叫张蛋蛋，其实原名是张寒薇，湖南人，现在应该是在重庆。弹吉他的姐姐，好喜欢。很有感觉，哈哈哈。

继续弹吉他啦，也许以后会有机会见面的哦！

整理一下你自己的样子啦，现在这位中年油腻大叔太难看了。你是一个漂亮的小哥哥啊。帅气，阳光一点。要么怎么和弹吉他的姐姐搭呢。

做自己喜欢的自己，做最帅最靓的仔。
