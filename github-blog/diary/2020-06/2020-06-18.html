# 2020-06-18

像往常一样，今天早上起来的第一件是就是去收集蚂蚁森林的能量，这已经是我一个多月以来一直在做的一件事情了。其他的都没有坚持住。

之后我去煮面条，当做自己的早餐，这也是最近早餐的标配了。因为自己不上班，妈妈去工作的地方吃饭，爸爸去值夜班，早上饭在工作的地方吃过了。

我吃过早饭，心情特别不好，我也怀疑自己这么多年来是一直被抑郁症困扰着。很少有什么高兴的地方。一方面是我不够强大到战胜它。因而给这个家带来了很多的困难。现在真的是最困难的时期了。没有工作就没有收入。生活在城市里面。是时时刻刻要花钱的。当务之急，是要早点上班，让一切都慢慢变好。

我之前也很认同郦波讲的曾国藩日课十二条。去喜马拉雅买了音频来听，那几天是很有收获，但是时间一长。又回到原来的样子了。

独处的这段时间，我实在是浪费了很多的时间，嗜睡，抽烟，打飞机，东看西看各种视频和文章，胡乱的去想自己未来的各种事情。当然这些个东西，都加重了我的精神负担。金钱的负担，都是由父母去解决。活脱脱的像个啃老族，一点都不争气。

在责怪自己的时候，也不能激起自己的愧疚感使得自己坚强起来。

今天又一次看到了曾国藩说的，人但有恒，事无不成。打心底是认同他的这种观点的，所以还想着再去试一试，给自己一些希望的东西。

于无声处听惊雷。

不在沉默中爆发，就在沉默中消亡。去吧，相信未来。每天来这里和我相会，我会一直都在。

懦弱无刚乃男人最大之耻。--曾玉屏

刚毅--曾国藩

昨天看到在美国的街头，有很多的流浪者，即使看上去那些很体面的人。他们没有工作，他们的失业率很高。为什么不他们不去工作，因为他们说自己不被雇主接受，从而找不到工作。我相信大部分流浪者是善良的，但也未必，因为其中有很多是瘾君子，因为吸毒而变得贫困，家庭破裂的事情太平常了。以后遇到流浪者，还是要给予施舍的。即使自己是那么贫困。但是我有一双可以创造价值的双手。在美国生活的中国夫妇，他们在中国有车有房，却非要去美国住那种7平米的房间。也许他们以后会过得好起来，因为我舅舅说他们也是住过狭小的地下室。现在能住上联排别墅，真心的祝福他们。

还有就是，我认为英语这个语言其实也很简单，看明道《冒险王》就真正能体会到，语言是用来用的，不管你的语法有什么错误，被人理解，传达意思，这才是第一性原理。即使在美国生活的人，他们说的英语，也真的很简单，单词是那么的常用。这可能和我现在长时间不说话，有关系。因为没有沉浸在母语的环境中。从而觉得英语也是很简单的。如果我住在舅舅家，当时跟着舅妈学英语，那应该是很棒的。所以现在多想想自己的可能性。毕竟人只有一辈子可以活。只有一次选择的机会。为什么要这么的不珍惜呢。

在日本的拾荒老爷爷，年龄看上去不小了，我觉得那是很真实的日本拾荒老人的写照，每天要比别人早起3个小时，去沿街搜寻易拉罐。依靠每天找废品赚来的8美元来支付自己的生活费。住在大街上。这样的生活，是存在的。只是我现在才看到而已。

在日本还有一部分人，他们是年轻人，但是他们选择了整天在狭小的房间里聊天、看视频、利用无聊的东西来打发时间。真的很丧，很难理解。但是他们每天真的要花掉好多钱，即使每天在那样狭小的房间里，每天还要支付将近130元人民币，在我们中国，130元都可以去住宾馆里。当然要是去租房子，还能省下更多。她们待的地方是东京，所以是那么贵。他们仿佛是被什么东西困住了，就像现在的我一样。真的太可惜了。还有那看上去穿戴光鲜，但是满屋垃圾的女人，真的很让人费解。这可能也是因为习惯的原因吧。所以真的不要忽略小小的恶习，因为它会在你不知不觉中长大，直到你被它所控制。

还有就是昨天晚上想到，为什么现在自己那么依靠手机，能不能不用手机，放下手机，放下电子产品。在现在，很难，以为要打卡，要付钱，要和同事联系。以至于大妈大爷也需要智能手机。我想放下手机，放弃那些我真正不需要的东，但是这些东西往往是我在不知不觉中堆积的。我算不上是一个简单的人，因为我的东西真的挺多的。其实人活着真的需要很少的东西就可以了。一个背包就可以装下。我想我以后在买东西的时候，一定要三思而行。不可以被感情操控。这确实是一个挑战。但是真的很有意义。因为简约的生活方式，真的很棒。也会为你省下很多钱。

我应该去工作，为了自己能开心一些，为了父母能少劳累一些。为了能早点还清自己的债务。为了世界上那些真正需要帮助的人。
