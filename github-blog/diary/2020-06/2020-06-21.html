# 2020-06-21

甚晏起，10点钟起床的，鸡汤面条。

打开电脑，进入哔哩哔哩，开始看《最后的生还者2》剧情很棒，画面优美，可是我没有机器可以上手。无法体验。没有钱，有的时间。我没有去工作，也就没有钱去做什么事情。没有钱，装修停止了。没有钱，我们一个月吃一次大餐，没有钱，爸爸坏掉的剃须刀，还一直缺少一个刀片。没有钱，父母被奴隶着，爸爸会上夜班，伤身体，体重已经是112斤了，我现在应该有130斤了吧。妈妈在餐馆上班，还要去打扫厕所。而我每天都是在电脑跟前看视频。无所事事。我每次坐在电脑前，都不去学习，这样混日子，自己也快要迷失了。

之后再看了《闪灵》，我有很长很长时间不去看恐怖片了，长的像是十年的样子。因为确实害怕。那些血腥的画面其实并不恐怖。恐怖的是内心的某些东西。

《闪灵》主人公是个面临中年危机的兼职作家，就像小说原著一样的职业。也多少有作者的生活写照。他们需要独自在一个大雪封山的旅馆单独住上5个月，那是很长的时间。杰克只会写作，没有乐趣，呆若木鸡。将自己逼疯掉了。我害怕自己也会把自己逼疯掉。这真的会是有可能的。精神分裂也是有可能的。

我无法去学习，应该是习惯的原因。我已经习惯了，打开电脑第一件事不是去看书，而是去看视频。就是这样。我认输了，我无法改变。我真的无法改变了。更不要说做圣人了。日课十二条，那太遥远了。那太遥远了。

之后的一步电影是《极速车王》，我对车感兴趣吗？我对极速感兴趣，那种疯狂的感觉才是棒的。

明天马上就要来了。明天马上就要过去了。我还是像昨天一样活着。

爱你，宫庆鑫。
