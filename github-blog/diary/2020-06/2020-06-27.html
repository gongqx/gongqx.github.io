# 2020-06-27

早上没有动力起床，也没有早起和晚起的分别了，那就说甚晏起。

心心念念的去浙大，去哪个校区呢，紫金港是最新最大的校区，5号线直通的。玉泉校区是计算机学院所在的校区，也是最老的校区，在西湖区。于是我决定去玉泉。

长途跋涉，近在眼前，匾牌庄严。有心中想象的美好模样。疫情期间，外人免进。

真是打击到了姥姥家了。

去学校附件兜兜转转，民宿，植物园，饭馆，男男女女。而我看起来就像是小学生。而我已经没有机会去就读。我已经大出他们五六岁。

时光荏苒，没有可以再浪费的机会了。

口袋空空，回家，乘车。

回程时，遇一中学女孩，地理试卷，将题干内的重点信息标出，快速提取试题眼。

煮面，吃饭，一人食（）。

孤独，泡澡温蛙（看B站）。

读书节目，买书之人海了去了。自己的那点就不用提了。

世界读书日，618年中大促，都是好时候。

智能路障（UP主）有点东西的人。

今天回家，煮了点面吃，一边吃一边哭，泪水滴落在碗里，没有开灯。——关灯吃面。

这是写作，很简单的一句话，很有气氛。很到位。

关于图书，我认为阅读也是要分级别——要先懂得基础知识，才能得到高级图书内的知识。

读书还要有分类，我应该先考试再说别的。

乱的像一锅粥的人，一天是不可能变成正常人的，更不要说是变成超人。

读书要有收获，要有记录，读太多书没有用。读太少书更是不行。

读书是为了什么，多思考多行动。

如果是个loser，没有change，那就可能永远是个loser。

不应该把读书看做是一种负担。那么美好的东西，被你用错误的方法，变得那么累。真是天底下最悲哀的事情了。

就像是现在的自己有那么多账要还，还不挣钱一样，令人抓狂。

一个基本的生活保障都没有哦。行动吧，行动吧，行不行。这一步怎么那么难迈，难以想象。这里有魔咒，能量异常的强大。其实就是总是在逃避。逃避。无法正面面对一切。这就是内心真是的样子。
