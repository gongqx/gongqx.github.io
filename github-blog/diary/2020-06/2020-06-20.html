# 2020-06-20

让人安心的是，花棒中下了。

上午十点多就去家博会了，在友邦谢康的帮助下，我签了两单，一单是友邦吊顶，一单是大自然木门。其实到了下午我才知道这是有套路的，打折就是那个几件东西。交了钱了，你不想在他们家做也没有办法了。以后还是要这样，三思而后行。多想想，拿出第一性原则来，其实内在的利益联系就是那么的简单，别被外在的表象所迷惑到就好。你会慢慢的成长的，因为你已经在吾日三省吾身了。别去在意别人，先做好自省。

真的很庆幸自己没有傻到签很多单。和那些有钱的本地人比起来。我的这几个小钱，真的是没有多大油水。但是对于我的来说，还是不小的一笔数字。展会的赠品就是那么便宜，这样的场所才真正能看出人是多么的喜欢贪小便宜。我自己也是其中之一位。

门是定下了，吊顶也差不多定下了。装修还是要接着进行。叔叔婶婶也一直在提醒，因为明年是一定要住进去的，装修好之后还要花上半年的时间去除味。所以时间不是很多了。总是在一些时刻，时间过得是那么快，不容易轻易抓住。

哦，几天发现自己说话的时候都说的不利索了，因为长时间没有说话的习惯的原因。所以变得迟钝了，幸好会场太吵，以至于别人也听不清楚我在说什么。哈哈哈。

还有就是 ExpressVPN 真的完成退款了，以后有钱就支持一下这个VPN。
